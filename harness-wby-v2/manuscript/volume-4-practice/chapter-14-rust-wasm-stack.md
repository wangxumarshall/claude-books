## Step 1: ts-rs跨语言对齐

```rust
use ts_rs::TS;
use serde::Serialize;

#[derive(Serialize, TS)]
#[ts(export, export_to = "bindings/")]
pub struct AgentConfig {
    pub name: String,
    pub max_iterations: u32,
    pub timeout_ms: u64,
}
```

## Step 2: WasmEdge部署

```bash
# 编译为WASM
cargo build --target wasm32-wasi

# 使用WasmEdge运行
wasmedge run --dir /data:/data agent.wasm
```
